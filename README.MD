#InstaHub

## Projeto

Um pequeno projeto em react native que utiliza a api do github para pesquisar o usuario e exibir seus projetos e seguidores dentro da plataforma.

## Tecnologias

- React Native
- React Navigation
- React Hooks
- UseContext
- Typescript
